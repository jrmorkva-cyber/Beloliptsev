# Пак правок по интервью с Белолипцевым — 23.06.2026

Готовый набор для исполнения правок сайта во **отдельной сессии Claude**. Каждая правка привязана к тайм-коду записи созвона и к кадру экрана в этот момент — что именно клиент видел и комментировал.

## Как пользоваться (быстрый старт)
1. Открой **`02-PROMPT-next-session.md`** → скопируй промпт целиком в новую сессию Claude (в папке сайта).
2. Claude работает по **`01-TZ-MASTER.md`** (все 57 правок, сгруппированы, с приоритетами).
3. По каждой правке есть кадр в **`screenshots/<ID>.jpg`** — визуальная сверка «то ли место».
4. Сначала 🔴 P0 (контент/SEO/баги), потом 🟡 P1 (визуал), 🟢 P2 — после проверки SEO-веса.

## Что внутри
```
00-README.md                 ← этот файл
01-TZ-MASTER.md              ← ПОЛНОЕ ТЗ: 57 правок, сводка по приоритету + по экранам
02-PROMPT-next-session.md    ← готовый промпт «закинуть и поехали»
03-edits-by-screen/          ← те же правки, разбиты по 9 экранам (по файлу на экран)
screenshots/                 ← 57 кадров из видео (по кадру на правку, имя = ID правки)
transcript/
  ├── interview-full.txt     ← полная расшифровка созвона
  └── timecodes.md           ← таблица: правка ↔ тайм-код ↔ кадр
_data/
  ├── edits.json             ← машинный источник правды (вся структура правок)
  ├── build-frames.mjs       ← нарезка кадров из видео по тайм-кодам (idempotent)
  └── gen-docs.mjs           ← генерация ТЗ из edits.json (DRY)
04-QA-REPORT.md              ← отчёт QA с виртуализацией (исполнимость без поломки прода)
```
