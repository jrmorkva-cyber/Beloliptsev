# START HERE — сайт Белолипцева + правки 23.06

В этом архиве две папки:
- **`site/`** — код сайта (Astro, статика). Это рабочая версия (та, что на `astro-seven-delta.vercel.app`).
- **`TZ-Interview-2026-06-23/`** — ТЗ правок с созвона с клиентом (57 правок + кадры + промпт).

## Пошагово
1. Открой **`TZ-Interview-2026-06-23/00-ДЛЯ-РИТЫ-прочти-первым.md`** — там всё расписано по-человечески.
2. В терминале:
   ```
   cd site
   npm install
   npx astro build      # проверить, что собирается (создаст dist/)
   ```
3. Запусти своего Claude **в папке `site/`** и вставь первым сообщением весь текст из
   **`TZ-Interview-2026-06-23/ПРОМТ-в-свой-Claude.md`**.
4. Дальше Claude идёт по правкам: сперва 🔴 P0, потом 🟡 P1. 🟢 P2 — не трогать.

## Важно
- Деплой/правки — только в этой `site/`. Это PortedPage-версия (страницы = `site/src/data/*.html`).
- К каждой правке есть кадр в `TZ-Interview-2026-06-23/screenshots/<ID>.jpg` — сверяй место по нему.
- Что непонятно или ждёт материала клиента (сертификаты, смежные секторы) — в общий чат.
