02 · Поздний Жолтовский и ведомственный дом на Садовом

Кватроченто на Смоленской

Дом построен в 1956 году по проекту Ивана Владиславовича Жолтовского для сотрудников ведомства государственной безопасности. Это поздний неоклассицизм с мотивами итальянского кватроченто — узнаваемая авторская манера мастера: строгий ордер, благородные пропорции, сдержанный, но точный декор.

Жолтовский всю жизнь опирался на классику Возрождения — и этот дом на Смоленской стал одним из его поздних высказываний, уже на излёте большой сталинской эпохи, рядом с высоткой МИД.

Квартиры строились крупными: просторные гостиные, по два санузла, парадные подъезды с художественной росписью. По меркам эпохи — жильё высокой категории для статусных жильцов.

Сегодня дом — объект культурного наследия. Авторство Жолтовского и охранный статус делают адрес особенным среди послевоенной застройки центра.
