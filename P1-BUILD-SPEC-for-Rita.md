## Сквозная структура

### G08 · блок контактов → в подвал по умолчанию
- **Где:** `sd-contact__coords` (email/тел) + `sd-contact__caps` (ЦИАН/Telegram/Офис) — напр. `uslugi.html` стр. 339–347; повторяется по страницам.
- **Что:** вынести этот блок в подвал (footer), чтобы после формы не было расфокуса (контакты→видео→меню). Проверить, что в футере не дублируется.

### G10 · единый фиксированный хедер, прозрачный на 1-м экране
- **Где:** `header.st-topbar` (есть на всех страницах, `visotki.html` стр. 68–86 — эталон) + CSS.
- **Что:** хедер `position:fixed`, ЕДИНЫЙ на всех. На 1-м экране (поверх hero-фото) — прозрачный минимал (лого + телефон + бургер, пункты прозрачным мелким). При скролле на 2-й экран → появляется плотная плашка (IntersectionObserver/scroll-Y). Не плодить лишние пункты меню на хабах.

### C01 · унифицировать бургер ↔ футер-меню
- **Что:** бургер-меню и футер-меню расходятся (где-то «зелёный» стиль) → равняться на синий/футер-стиль. Пункт сделать просто «**Сталинки**» (без приписки ЦАО) и перенести туда, где Премиум/Инвестиции (в разделы/сегменты). Сверить состав пунктов бургера и футера — должны совпадать.

### G09 · выравнивание текста по ширине (justify)
- **Где:** редакционные длинные блоки, на которые тыкал клиент: блоки траста на главной, «что включает сопровождение» (`uslugi.html`), описания на `stalinki.html`.
- **Что:** `text-align:justify` + где переносы рвут — `hyphens:auto`/`text-wrap` подгонять. НЕ применять к коротким подписям/заголовкам.

### G11 · упростить нижнюю сноску траст-факторов
- **Где:** `st-trust__tags` — строка-перечисление под блоками траста (напр. `modernizm--dom-narkomfina.html` ~324–327: «Белолипцев Владимир · РГР Топ-3 из 1080 · Аттестат «Эксперт» 19/1080 · ГРМ»). Повторяется на многих страницах.
- **Что:** оставить только **Топ-3 + одну ключевую регалию** (напр. «РГР Топ-3 из 1080 · Аттестат „Эксперт“ 19/1080»), лишнее перечисление убрать (расфокус режет конверсию). ⚠️ Сущности не терять полностью — если убираешь ГРМ/имя из сноски, они должны оставаться в другом траст-блоке страницы.
